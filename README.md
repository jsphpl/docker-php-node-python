# docker-php-node-python

**Tag:** `jsphpl/php-node-docker`<br/>
**Base Image:** `php:7.2-cli`

## A Docker image featuring
- php 7.2
- composer
- node 8
- npm
- yarn
- python 2.7
- pip

## Installed PHP extensions
- imagick
- zip
- bcmath
- gd
- exif
- pgsql
- pdo_pgsql
- xdebug (disabled by default, enable with `docker-php-ext-enable xdebug.so`)
